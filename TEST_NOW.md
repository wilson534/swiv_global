## ⚠️ 如果遇到问题

### 问题 1：看不到数据 / 显示错误
**解决：**
1. 检查 API 服务器是否运行：`http://localhost:3000`
2. 查看终端是否有错误
3. 确认 `.env.local` 文件存在且包含 OpenAI API Key

### 问题 2：AI 问答失败
**可能原因：**
- OpenAI API Key 无效或过期
- 网络连接问题
- API 配额用完

**查看终端错误信息！**
